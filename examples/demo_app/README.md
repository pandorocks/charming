# DemoApp

A generated Charming app extended with demo features:

- application model state
- command palette and modal overlay
- viewport-backed activity log
- screen-aware layout
- timer-driven spinner

Run it with:

```sh
bundle install
bundle exec exe/demo_app
```

Keys:

```text
up/down     change the counter
j/k         scroll the activity log
p           open the command palette
q           quit
```
